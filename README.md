# OpenInfra

A infrastructure project for a typical tech company.  This project includes OSS projects for:

* Identity Management
* Project Management
* CI/CD
* VCS
* Prototyping
* Docker registry

## Quickstart

* Run `bin/start`
* Follow instructions printed after the start command completes (update /etc/hosts)

## Prereqs

* kind
* terraform
* helm
* kubectl
